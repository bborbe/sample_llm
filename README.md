# Pydantic AI Samples

## Pyenv

```
pyenv virtualenv 3.12.4 sample_pydantic_ai
pyenv local sample_pydantic_ai
```

## Install deps

```
pip install -r requirements.txt
```

## Ollama

Install ollama:
https://ollama.com/

Install model: 
```
ollama run gemma2:9b
ollama run llama3.1:8b
ollama run llama3.2:3b
ollama run mistral:7b
```
